## Supported OS

1. `macOS 26.0+`

## What's New

1. Public Preview release channel and release packaging policy.
2. Middle-button drag scrolling, smooth wheel behavior, and mouse button action controls.
3. Manual update flow documentation and release checksum publishing workflow.

## Accessibility Requirement

1. Mouse/scroll interception requires Accessibility permission.
2. In-app path: `General > Permissions > Request…`
3. System path: `System Settings > Privacy & Security > Accessibility`

## Installation

1. Download ZIP from GitHub Releases.
2. Unzip.
3. Move `MouseManager.app` to `/Applications`.
4. First launch with right-click `Open`.
5. If blocked, click `Open Anyway` in `Privacy & Security`.

## Known Limitations

1. Public Preview builds are distributed without Developer ID signing/notarization.
2. macOS 26.0+ only.
3. Updates are manual (no in-app auto-update yet).

## Update Policy

1. Manual updates only.
2. Download the latest release ZIP and replace `MouseManager.app` in `/Applications`.
