## Accessibility Requirement

1. Mouse/scroll interception requires Accessibility permission.
2. In-app path: `General > Permissions > Request…`
3. System path: `System Settings > Privacy & Security > Accessibility`

## Installation

1. Download ZIP from GitHub Releases.
2. Unzip.
3. Move `MouseManager.app` to `/Applications`.
4. First launch with right-click `Open`.
5. If blocked, click `Open Anyway` in `Privacy & Security`.

## Known Limitations

2. macOS 26.0+ only.
3. Updates are manual (no in-app auto-update yet).
